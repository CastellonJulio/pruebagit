Para iniciar
1 - git init
Para agregar archivos o todo 
2 - git add .
Para ver el estado y los archivos que se quieren agregar
3 - git status
Eliminar alguna carpeta que ya no quiera subir
4 - git rm --cached (nombre del archivo)
Comentar cada cambio que se realiza
5 - git commit - "mensaje"
restaurar un cambio
6 - git restore <nombre del archivo archivo>

Paracrear una rama de master
7 - git branch clonmaster
Ir a la rama creada
8 - git checkout clonmaster
Para unir ramas
9- git merge <nombre de la rama>
